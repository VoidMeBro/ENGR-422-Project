for me, first run:
cd "Backend - Database connection"   --- this is to go to backend where I will try to make things work and connect to MySQL

run the following (unless its already done and the Package and Package-lock files are already made
1.  npm init -y
2.  npm install express cors nodemon

	(express is the node.js framework required to connect to the MySQL)
	(cors is required to access the API in the react tab)
	(nodemon refreashes the server when changes are made)

3. npm install express cors mysql2 dotenv 

	(Due to using typescript, mysql2 and dotenv are extra runtime dependancies and libraries to make typescript work)

4. npm install --save-dev typescript ts-node nodemon @types/express @types/cors @types/node

	(this creates Type definitions to make Typescript work with express, cors, and node.js)



.env --- holds the data for the database that will be connected to as follows:

DB_HOST=localhost 	(who hosts the database)
DB_USER=root		(username 
DB_PASSWORD=		(password)
DB_NAME=		(name of the database)
DB_port= 		(port of XAMPP to be used)
