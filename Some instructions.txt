Connect to MYSQL using XAMPP
(----------------------------------)

click on "Shell on the right side"
1.Run the following codes in the new terminal:
	mysql -u root
   	ALTER USER 'root'@'localhost' IDENTIFIED BY 'PSUMySQL';
   	FLUSH PRIVILEGES;
   	EXIT;

2. go to phpMyAdmin folder in XAMPP (in my case C:\Users\USER-PC\Development\xampp\phpMyAdmin)

3. open file config.inc.php

4. change the following lines
   $cfg['Servers'][$i]['user'] = 'root';
   $cfg['Servers'][$i]['password'] = 'PSUMySql'; // change to the password of the database

   $cfg['Servers'][$i]['controluser'] = 'pma';
   $cfg['Servers'][$i]['controlpass'] = 'PSUMySql'; // change to the password of the database

5. on XAMPP the Admin button should work. 
(note, at this point you will only have a user called "root")
(apaarently this should be a functional state, but to access more advances features you should make a "pms" user)

to do so, do the following;
(optional, apparently)

6.Open MySQL

7. run the following code:   
	CREATE USER 'pma'@'localhost' IDENTIFIED BY 'PSUMySql'; //(change password to whatever. I keep it same for simplicity)
   	GRANT ALL PRIVILEGES ON `phpmyadmin`.* TO 'pma'@'localhost';
   	FLUSH PRIVILEGES;
8. for the remaining error, just click "Find out why" and click something in the line of "Create".
(it should automatically configure itself)
